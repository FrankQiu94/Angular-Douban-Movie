# Angular-Douban-Movie
A simple Demo of Douban Movie in PC by AngualrJS 1.0

###打开方式
在该文件夹中打开Terminal，输入
```bash
browser-sync start --server --files \"*.html, css/**/*.css, app/**/*.html, app/**/*.js\"
```
或
```bash
npm run start
```
通过browser-sync自动打开
```bash
http://localhost:3000
```
服务端，运行
```bash
index.hmlt
```
